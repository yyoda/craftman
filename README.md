# 週次レポート自動化プロジェクト (Project Alpha)

営業チーム向けの週次レポートを自動生成するプロジェクトです。

## プロジェクト概要

- **目的**: 営業チーム（20名）向けの週次レポート作成を自動化する
- **対象読者**: 社内営業部

## ディレクトリ構成

- `inbox/`: 営業メンバーから届く生データ（CSV、Excel）を格納します。**この中のファイルは絶対に削除・変更してはいけません。**
- `drafts/`: 作業中のレポート一時保存場所です。
- `outputs/`: 完成したレポートが出力されます。ファイル名は `YYYY-MM-DD-weekly-report.xlsx` の形式です。
- `archive/`: 先週以前の完了分を保管します。
- `references/`: テンプレートやガイドラインなどの参照資料を格納します。

## スタイルガイド

- **文体**: 敬体（です・ます調）で統一してください。
- **数値表記**: 
  - 数値は千の位にカンマを入れる（例: 1,000）
  - 通貨は `¥` を前置する（例: ¥1,000）
- **フォント**:
  - 本文: 游ゴシック
  - 見出し: Noto Sans JP Bold

## 制約事項

1. **データ保全**: `inbox/` 内の元データファイルは絶対に削除・変更しないこと。
2. **ファイル命名規則**: `outputs/` のファイル名は `YYYY-MM-DD-weekly-report.xlsx` とすること。
3. **プライバシー**: 個人名は匿名化すること（例: 山田太郎 → メンバーA）。
