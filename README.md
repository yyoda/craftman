# ドキュメント生成エンジン (Project Craftsman)

様々な入力データをもとに、高品質なドキュメント（レポート、スライド、仕様書等）を自動生成するエンジンです。

## プロジェクト概要

- **目的**: 定型的なドキュメント作成業務を自動化し、クリエイティブな業務への集中を支援する
- **主な機能**:
  - 多様な入力フォーマットへの対応 (CSV, JSON, Excel, Markdown)
  - マルチフォーマット出力 (PDF, Excel, PowerPoint, Word)
  - テンプレートベースの柔軟なデザイン適用

## ディレクトリ構成

- `inbox/`: 生成リクエストとなる入力データを格納します。ここにファイルが追加されると処理が開始されます。
- `drafts/`: 生成途中または承認待ちのドキュメントが一時保存されます。
- `outputs/`: 最終的な成果物が出力されます。
- `archive/`: 処理完了後の入力データおよびログを保管します。
- `references/`: 各種テンプレート、スタイルガイド、設定ファイルを格納します。

## スタイルガイド & 制約事項

1. **データ保全**:
   - `inbox/` 内の元データは、処理完了後に自動的に `archive/` へ移動されます。手動での削除は推奨されません。

2. **出力仕様**:
   - ファイル名は入力データ内のメタデータまたは設定ファイルに基づいて自動決定されます。
   - デフォルトでは `YYYY-MM-DD-[type]-[name].[ext]` の形式が適用されます。

3. **デザイン基準**:
   - [ブランドガイドライン](references/brand-guide.md) に準拠した配色とフォントを自動適用します。
   - 数値表記や日付フォーマットは、テンプレートごとのロケール設定に従います。
