# エージェント構成 (Project Alpha)

本プロジェクト「週次レポート自動化」において稼働する自動化エージェント（ボット/スクリプト）の定義と役割記述です。

## 1. Data Watcher (データ監視エージェント)
- **役割**: `inbox/` フォルダを常時監視し、営業メンバーからのデータ提出を検知する。
- **主なタスク**:
  - ファイル形式の検証（CSV/Excel）
  - データのクレンジングと標準化
  - 必須項目の欠損チェック
  - 不備があった場合の差し戻し通知

## 2. Report Generator (レポート生成エージェント)
- **役割**: クレンジングされたデータをもとに、実際に提出用のレポートを作成する。
- **主なタスク**:
  - データの集計・分析
  - `references/report-template.xlsx` をベースにした帳票作成
  - スタイルガイド（ブランドカラー `#004E89`, `#FF6B35` 等）の適用
  - 匿名化処理（個人名マスキング）
  - 生成物を `drafts/` に保存

## 3. Delivery Bot (配信エージェント)
- **役割**: 完成したレポートを適切なチャネルで配信する。
- **主なタスク**:
  - `drafts/` のレポートに対する承認フロー管理（将来実装）
  - 承認済みレポートの `outputs/` への移動とリネーム（`YYYY-MM-DD-weekly-report.xlsx`）
  - 関係者へのメール/チャット通知
  - 処理完了後の元データの `archive/` への移動

## 将来の拡張構想 (Future Agents)

- **Insight Analyst (分析エージェント)**:
  - レポート内容から特筆すべき傾向（売上の急増・急減など）をAIで分析し、サマリーコメントを付記する機能。
